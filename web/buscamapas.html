Descripción del proyecto

Esta aplicación web permite explorar mapas interactivos de distintas ciudades, mostrando puntos de interés como plazas, iglesias, estadios y museos, cada uno identificado con un icono específico. Además de seleccionar ciudades predefinidas, el usuario puede buscar cualquier ubicación para añadir marcadores personalizados. La app también integra información adicional en tiempo real, como datos sobre terremotos y otros eventos relevantes, ofreciendo una experiencia dinámica y completa. Incluye una leyenda interactiva para facilitar la comprensión de los símbolos en el mapa.

Especificaciones técnicas

El proyecto utiliza la API de Google Maps para renderizar el mapa y administrar los marcadores. Cada ciudad tiene un conjunto de ubicaciones definidas con nombre, coordenadas y un icono específico que representa el tipo de lugar (plaza, iglesia, estadio o museo). Al seleccionar una ciudad, el mapa centra su vista en la ubicación correspondiente y ajusta el nivel de zoom. Los marcadores se generan dinámicamente y se almacenan para facilitar su eliminación cuando se cambia la ciudad. La leyenda se actualiza automáticamente con los iconos y categorías presentes y puede ocultarse o mostrarse mediante un botón. Se manejan eventos DOM para controlar la interacción con el selector de ciudad y el botón de la leyenda, asegurando una experiencia de usuario fluida y responsiva.
